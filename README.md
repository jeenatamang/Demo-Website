# EduNepal Frontend

A simple React frontend for EduNepal with pages like Home, About, Courses, Events, Blog, and Contact.

## Features

* Navigation with React Router
* Multiple pages (Home, About, Courses, etc.)
* Clean UI

## Tech Stack

* React
* JavaScript

## Installation

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install
npm run dev
```
